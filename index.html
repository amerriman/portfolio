FLASH CARDS

An interactive study application that allows users to choose from ready-made study sets, or create their own custom flash cards. Version 1 was completed after six weeks in the FullStack program and utilized only Javascript, jQuery, HTML5 and CSS3. Current version was refactored to include Node.js, Express and MongoDB.

Technologies: HTML5, CSS3, Bootstrap, JavaScript, jQuery, Node.js, Express, MongoDB


TRANSLATORSAURUS

Language translation app that utilizes the Bing Translator API to provide translation in more than 20 languages with speech-language capabilities included. Users can quiz themselves with a 20 question practice test that automatically resets after five incorrect answers. Built in collaboration with Keith Hopkins, Rob Hajek, and Ryan Ciecalone.

Technologies: HTML5, CSS3, Bootstrap, JavaScript, jQuery, Node.js, Express, MongoDB, Bing Translator API, speech-language audio


WINE MATCH

Wine Match is a MEAN stack application that that utilizes a RESTful API to help the user choose the perfect wine for a meal. With in-depth and real-time information obtained via the Snooth API, users can sort through an extensive list of wines, and associated details, including: recipes, ratings, user reviews, and winemaker notes. Future version in progress and includes refactoring for the Hot Towel model.

Technologies: HTML5, CSS3, Bootstrap, JavaScript, Node.js, Express, MongoDB, Angular, Mocha & Chai testing, Gulp, Passport, Snooth API


WriTone

WriTone uses the MEAN stack and a RESTful API combined with the power of the Watson Alchemy API to help teachers and students analyze the tone of their writing. 'Teacher' users' main dashboard view showcases NV.D3 charts and responsive design that allows for an intuitive user experience. 'Student' users see a simplified dashboard with access to edit and delete their writing submissions, as well as view NV.D3 charts to track their own performance. Basic analysis functionality persists for non-logged in users.

Technologies: HTML5, CSS3, Bootstrap, Satellizer, Travis CI, NV.D3, JavaScript, Node, Express, MongoDB, Angular, Gulp, Alchemy API, Heroku
